# Project Name
* AkanNames

## Author

### Description 
* This project tries to calculate the day of the week someone was born and gives you the Akan name according to your day of birth.
   This was naming system mainly used in Ghana and many prominent leaders like koffi Annan have been named through this system.

 #### Set up instructions
